# راهنمای اجرای پروژه در سیستم شخصی (Local)

این پروژه با استفاده از **React** و **Vite** ساخته شده است. برای اجرای آن، شما نمی‌توانید صرفاً فایل `index.html` را در مرورگر باز کنید. باید مراحل زیر را دنبال کنید:

## ۱. پیش‌نیازها
ابتدا باید **Node.js** را روی سیستم خود نصب داشته باشید. می‌توانید آن را از [nodejs.org](https://nodejs.org) دانلود و نصب کنید.

## ۲. راه‌اندازی در PHPStorm
۱. تمامی فایل‌های پروژه را در یک پوشه قرار دهید.
۲. پوشه را با **PHPStorm** باز کنید.
۳. در پایین محیط PHPStorm، برگه **Terminal** را باز کنید.

## ۳. نصب پیش‌نیازها (Dependencies)
در ترمینال، دستور زیر را تایپ کرده و اینتر بزنید تا تمام کتابخانه‌های مورد نیاز نصب شوند:
```bash
npm install
```

## ۴. اجرای پروژه
پس از اتمام نصب، دستور زیر را برای اجرای سرور توسعه اجرا کنید:
```bash
npm run dev
```

## ۵. مشاهده نتیجه
پس از اجرای دستور بالا، یک لینک (معمولاً `http://localhost:3000`) در ترمینال ظاهر می‌شود. با کلیک روی آن یا کپی کردن آن در مرورگر، می‌توانید پروژه را مشاهده کنید. هر تغییری که در کدها در PHPStorm بدهید، برنامه به صورت خودکار در مرورگر به‌روزرسانی می‌شود.

---

### چرا فایل index.html به تنهایی کار نمی‌کند؟
این پروژه از ماژول‌های مدرن JavaScript و سیستم TypeSript استفاده می‌کند که مرورگرها به تنهایی قادر به تفسیر مستقیم آن‌ها نیستند. ابزار **Vite** این کدها را پردازش کرده و به زبانی تبدیل می‌کند که برای مرورگر قابل فهم باشد.
